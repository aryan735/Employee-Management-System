## **âš¡ Getting Started**  

### **1. Clone the Repository**  
```bash
git clone https://github.com/aryan735/Employee-Management-System.git

```

### **2. Configure Database**  
- Create a **MySQL database**:  
```sql
CREATE DATABASE employee_db;
```
